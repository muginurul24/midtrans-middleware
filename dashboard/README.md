# Dashboard

Dashboard operator dan store owner untuk `payment-platform`.

## Stack

- React `19`
- Vite
- TypeScript
- Tailwind CSS v4

## Menjalankan Lokal

```bash
cp .env.example .env
pnpm install
pnpm dev
```

Default env:

```env
VITE_API_BASE_URL=http://localhost:8080
```

## Command

```bash
pnpm dev
pnpm build
pnpm lint
```
